## 2. MVP 完成结论

按当前 `PLAN.md` 的里程碑 4 定义，以下要求已经满足：

1. 保留 manifest、Application Support 本机模型包加载、模型资源预检和 Core ML 运行时接口审计。
2. 保留 tokenizer、图片预处理、向量持久化、模型版本记录、旧向量需重建标记和余弦相似度检索基础链路。
3. 保留技术验证报告、验证摘要和视觉索引/查询门禁。
4. 模型未配置、未就绪或未通过当前模型版本技术验证时，视觉索引和视觉查询会明确降级，不会生成伪视觉向量或伪视觉候选。
5. App 不下载模型、不上传图片、不使用人工标签、OCR 文本、文件名或基础元数据冒充视觉语义搜索。
6. 后续第 5、6、7 里程碑可以先依赖 OCR、时间和类型信号继续交付 MVP，不再被真实视觉模型阻塞。

因此，里程碑 4 的 MVP 结论是：**自动验收通过，真实视觉语义搜索作为增强能力延期。**

## 3. 后续增强项

以下事项不再作为里程碑 4 MVP 的完成门槛，移入真实视觉语义搜索增强阶段：

1. 接入真实可运行的 Core ML 图文双塔模型。
2. 确认图片编码器、文本编码器和 tokenizer 来自同一向量空间。
3. 用 5 张真实 Photos 样本验证图片向量和文本向量维度一致。
4. 验证相关查询相似度高于无关查询。
5. 分别验证中文、英文和中英文混排查询效果。
6. 记录真实模型的文件大小、许可证、加载耗时、单图处理耗时和资源占用。
7. 完成真实视觉索引批处理的人工验收。
8. 证明画面类自然语言查询能够返回真实语义相近图片。
9. 让视觉信号参与后续融合排序。

这些增强项只有在合法模型、匹配 tokenizer、预处理配置和真实样本技术验证全部通过后才能启用。

## 6. 尝试过但失败或未继续的路线

### 6.1 使用 `xcodebuild test` 完整跑 XCTest

失败原因：当前沙箱无法连接 `com.apple.testmanagerd.control`。已用直接运行 XCTest 的方式完成当前环境可执行的测试验证。

### 6.2 仅检查资源存在就认为模型可用

失败原因：测试发现无效 `.mlmodelc` 目录可能绕过过浅的资源存在检查。随后补充 Core ML 运行时审计，让 readiness report 检查模型能否被 Core ML 加载、输入输出是否匹配、shape 和数据类型是否符合配置。

当前状态：该问题已修正，并由测试覆盖。

### 6.3 下载或替换真实模型

未继续原因：用户明确要求不要下载或更换模型；项目规则也要求新增外部模型或依赖前先说明来源、许可证、替代方案和影响。

当前状态：没有新增网络模型，没有加入外部依赖。

## 7. 当前完成边界

在不下载、不更换模型、不扩大改动范围的前提下，当前代码已经把第 4 里程碑 MVP 需要的最小骨架打通到以下程度：

1. App 能知道视觉模型是否配置、是否可加载、是否通过验证。
2. 未配置或未验证时，视觉能力会明确降级，不会伪装成可用。
3. 真实模型放入本机模型包后，可以沿现有 manifest、tokenizer、Core ML 审计、技术验证、索引写入和余弦检索链路继续推进。
4. 后续里程碑可以先依赖 OCR、时间和类型信号继续开发 UI 和融合排序骨架；视觉信号在未通过真实验证前保持不可用。

## 8. 后续增强的最小下一步

如果后续要启用真实视觉语义搜索，最小下一步不是继续写大量业务代码，而是准备一个合法、可本机运行的 Core ML 图文双塔模型包，并通过现有验证门禁。

建议最小操作：

1. 选择一个许可证可接受、体积可接受、能在当前 macOS/Xcode 运行的 CLIP 类图文模型。
2. 在本机准备模型包目录：

```text
~/Library/Application Support/PictureSearch/EmbeddingModelPackage/
```

3. 放入以下文件：

```text
EmbeddingModelManifest.json
CLIPImageEncoder.mlmodelc
CLIPTextEncoder.mlmodelc
clip_tokenizer.json
```

4. 在 manifest 中填写真实模型来源、许可证、大小、SHA-256、输入输出名称、向量维度、tokenizer、图片尺寸和归一化配置。
5. 准备 5 张真实 Photos 样本对应的 `EmbeddingValidationSamples.json`，覆盖中文、英文和中英文查询。
6. 运行现有模型 readiness 检查和技术验证。
7. 只有技术验证通过后，再启用小批量视觉索引和真实语义查询验收。
