# 视觉模型候选评估

本文用于里程碑 4「本地图文向量与视觉语义搜索」的模型接入决策。当前评估只记录候选模型来源、许可证、替代方案和影响，不把任何外部模型、权重、转换脚本或新依赖加入工程。

## 当前结论

- 首选技术路线：使用本地 Core ML 图文双塔模型，图片编码器和文本编码器输出同一向量空间，通过余弦相似度做视觉语义检索。
- 首选候选：OpenAI CLIP ViT-B/32 或同等 CLIP 类模型的本地 Core ML 转换产物。
- 暂不作为默认内置模型：Apple MobileCLIP / MobileCLIP2 官方权重。
- 暂不采用：云端 embedding、人工标签、OCR 文本或文件名代替视觉 embedding。

## 候选 1：OpenAI CLIP

- 来源：OpenAI `openai/CLIP` 仓库。
- 许可证：仓库许可证为 MIT。
- 适配价值：CLIP 原生包含图片编码器和文本编码器，适合当前图文同空间向量检索目标；官方示例包含 `encode_image`、`encode_text` 和归一化后相似度计算，和当前工程的 `EmbeddingService`、`IndexStore.visualSearchCandidates` 方向一致。
- 接入要求：需要将图片编码器和文本编码器转换为可在 macOS 本地运行的 Core ML 模型；需要确认 tokenizer、文本序列长度、图片预处理、输入输出名、输出维度、模型大小和推理耗时。
- 主要风险：中文查询效果不一定可靠；转换后的 Core ML 输入 shape、数据类型和输出名可能与工程 manifest 不一致；模型体积和首次加载耗时需要实测。
- 当前状态：适合作为优先候选，但尚未加入模型文件，不能标记视觉语义搜索完成。

## 候选 2：Apple MobileCLIP / MobileCLIP2

- 来源：Apple `apple/ml-mobileclip` 仓库。
- 许可证：代码为 MIT；模型权重使用 Apple Machine Learning Research Model 相关条款。
- 适配价值：面向移动和本地推理优化，模型体积、参数量和延迟指标更贴近 macOS 本机 App 的性能目标。
- 阻断原因：官方模型权重条款限定 Research Purposes，且明确不包括商业产品或服务使用；因此不能作为默认可用模型随 App 内置。
- 可保留用途：可作为研究和性能参考，或在用户自行提供且确认许可证适用的前提下进行本机技术验证。
- 当前状态：不作为默认模型接入路线。

## 候选 3：OpenCLIP 生态模型

- 来源：`mlfoundations/open_clip` 及其兼容模型生态。
- 许可证：OpenCLIP 代码为 MIT；具体模型权重许可证需要逐个确认，不能用代码许可证替代权重许可证。
- 适配价值：模型选择多，可能存在更适合中文或多语言查询的图文模型。
- 主要风险：权重来源、训练数据、tokenizer 和许可证差异较大；转换、评测和维护成本高于单一 OpenAI CLIP 路线。
- 当前状态：作为备选路线。只有在具体权重许可证、tokenizer、体积和效果都确认后，才能进入 Core ML 转换和 5 张真实样本验证。

## 明确不采用的替代方案

- 云端 embedding：违反默认本机处理和不上传图片、OCR 文本、向量的隐私边界。
- 人工标签核心搜索：违反“不使用人工标签代替视觉搜索”的里程碑约束。
- OCR 文本、文件名或基础元数据模拟视觉搜索：只能补充文字、时间或类型信号，不能证明画面类查询能力。
- 直接读取 Photos Library 文件包批量导出图片训练或转换：违反 PhotoKit 读取边界。

## 加入真实模型前的最小准入清单

1. 记录模型来源、权重来源、转换来源和转换步骤。
2. 记录代码、权重、tokenizer 和转换产物的许可证。
3. 记录替代方案和未采用原因。
4. 记录模型文件大小、SHA-256、tokenizer 文件大小和 SHA-256。
5. 在真实 `EmbeddingModelManifest.json` 中填写 `integrationReason`、`alternativesConsidered` 和 `expectedImpact`。
6. 通过 `EmbeddingModelPackageInspector` 预检，确认 `.mlmodelc` 可加载、输入输出名存在、输入 shape 和数据类型与 manifest 一致。
7. 使用至少 5 张真实 Photos 样本完成本机技术验证，覆盖中文、英文和中英文混合查询。
8. 验证图片向量和文本向量维度一致，相关查询相似度高于无关查询并达到最小 margin。
9. 记录单张平均推理耗时、主要失败类型、中文效果限制和降级方案。
